# 喵星咖啡 — 实训项目介绍与演示指南

## 项目概述

《Web前端开发技术课程设计》实训项目，一个温馨可爱的**猫咖预约网站**。用户可以浏览猫咪信息、预约撸猫、点单咖啡饮品、参与限时秒杀活动、留言互动，还有 AI 助手为你答疑解惑。

## 实训八大任务实现详情

### 任务一：登录注册功能

**要求**：用户名、密码、手机号、邮箱做格式校验，实现普通用户和管理员角色登录注册

**实现要点**：

- **正则校验**（`lib/validators.ts`）：
  - 用户名：`/^\w{4,16}$/` → 4~16位字母/数字/下划线
  - 密码：`/^(?=.*[a-zA-Z])(?=.*\d).{6,}$/` → 至少6位，必须含字母和数字
  - 手机号：`/^1[3-9]\d{9}$/` → 1开头，第二位3~9，共11位
  - 邮箱：`/^[\w.-]+@[\w.-]+\.\w+$/` → 标准邮箱格式
- **角色选择**：注册页面支持选择"普通用户"或"管理员"角色（radio 单选按钮）
- **预设账号**：`demo/demo123`（普通用户）、`admin/admin123`（管理员）
- **登录状态**：全局管理（AuthContext），localStorage 持久化
- **错误提示**：校验不通过时输入框红色边框 + 红色背景 + 错误文字

### 任务二：菜单栏功能

**要求**：至少5个一级菜单，建议增加二级菜单

**实现要点**：

- **5个一级菜单**：首页、咖啡信息、猫咪信息、个人中心、留言板
- **二级菜单**：
  - 咖啡信息 → 全部商品/咖啡/饮品/甜品
  - 猫咪信息 → 全部猫咪/橘猫/布偶猫/英短
- **交互**：桌面端 hover 下拉展开，移动端折叠汉堡菜单
- **当前页高亮**：`bg-amber-100 text-amber-700`
- **右侧入口**：AI 助手、购物车（角标）、登录/注册

### 任务三：首页-轮播图功能

**要求**：可自动播放、上一页下一页手动切换

**实现要点**：

- `setInterval(next, 4000)` 每4秒自动切换
- 鼠标悬停暂停，离开恢复
- 左右箭头手动切换 + 底部圆点指示器
- CSS `translateX` + `transition` 平滑过渡动画
- 5张轮播图数据

### 任务四：猫咪信息-预约功能

**要求**：显示所有猫咪，选中显示详情，选择预约时间，点击预约显示成功提示

**实现要点**：

- 卡片式网格布局 + 搜索框（按名字/品种搜索）
- 动态路由详情页 `/cats/[id]`
- 详情页集成图片放大镜
- 预约表单：日期选择（禁止过去日期）+ 时间段选择（3个时段）
- 预约确认弹窗 → 写入 localStorage
- 未登录跳转登录页

### 任务五：咖啡信息-点单功能

**要求**：显示所有商品，选中显示详情，数量选择，购买功能

**实现要点**：

- 搜索框（按名称/标签搜索）+ 分类筛选（全部/咖啡/饮品/甜品）
- 商品详情页 `/menu/[id]`，集成图片放大镜
- 秒杀商品显示红色秒杀价+划线原价
- 数量选择器 + "加入购物车"
- 购物车：增删改查 + 总价计算 + localStorage 持久化

### 任务六：限时秒杀功能

**要求**：首页显示秒杀，点击跳转商品页面，参照京东秒杀

**实现要点**：

- 首页秒杀展示区（3件商品 + 倒计时）
- 独立秒杀活动页 `/seckill`
- 倒计时精确到秒（天:时:分:秒），`setInterval` 每秒更新
- 京东风格红色主题设计
- 秒杀状态：进行中 / 已结束

### 任务七：图片放大功能

**要求**：鼠标移动时，图片右侧显示放大后的对应位置

**实现要点**：

- `onMouseMove` 获取鼠标相对位置百分比 `(x%, y%)`
- CSS `background-image` + `backgroundSize`(2.5x) + `backgroundPosition`
- 鼠标进入显示，离开隐藏
- 仅大屏（lg 断点）显示，移动端隐藏
- 集成在猫咪详情页和商品详情页

### 任务八：留言板功能

**要求**：能正常显示留言信息

**实现要点**：

- textarea + 发表按钮，登录后可留言
- 留言列表展示（头像、昵称、时间、内容）
- 数据持久化到 localStorage
- 空状态提示"暂无留言，快来抢沙发吧！"

## 创新点

### 1. AI 助手（DeepSeek 大模型集成）

- 集成 DeepSeek API（`deepseek-v4-flash` 模型），实现聊天式 AI 交互
- **nginx 反向代理**保护 API Key，前端完全不可见
- **流式输出**：SSE 逐字显示，打字机效果
- **多会话管理**：创建/删除/切换聊天会话
- **防抖持久化**：聊天历史 500ms 防抖写入 localStorage
- **智能滚动**：流式输出用 instant 滚动，用户上翻时暂停自动滚动
- **人设设定**：猫咖店长小助手
- **停止生成**：AbortController 中断流式输出

### 2. 管理员角色与管理后台

- 注册时支持选择"普通用户"或"管理员"角色
- 导航栏管理员用户名旁显示紫色盾牌图标
- 个人中心"管理后台"选项卡：
  - **用户管理**：查看所有注册用户及角色
  - **预约管理**：查看所有预约，可确认/取消

### 3. 完整的订单系统

- 购物车结算 → 确认弹窗 → 生成订单
- 订单包含商品明细、总价、状态、时间
- 个人中心"我的订单"展示历史订单

### 4. 全站搜索功能

- 猫咪页：按名字或品种搜索
- 商品页：按名称或标签搜索
- 搜索与分类筛选联合过滤

### 5. 响应式设计

- 导航栏桌面端下拉 / 移动端折叠
- 图片放大镜仅大屏显示
- 移动端秒杀入口

### 6. CSS 驱动动画（零第三方动画库）

- 轮播图：纯 CSS translateX + transition
- 放大镜：纯 CSS backgroundPosition
- hover 缩放、卡片阴影均用 Tailwind

### 7. 现代技术栈

- Next.js 16 App Router（动态路由）
- TypeScript 严格模式
- Tailwind CSS 4
- GitHub Actions CI/CD

## 演示流程建议

1. **首页** → 展示轮播图自动播放、秒杀倒计时
2. **猫咪信息** → 搜索过滤 → 点击查看详情 → 图片放大镜 → 预约猫咪
3. **咖啡信息** → 搜索框搜索 → 分类筛选 → 加入购物车
4. **购物车** → 修改数量 → 结算 → 确认下单 → 下单成功
5. **限时秒杀** → 倒计时 → 点击跳转商品详情
6. **留言板** → 登录后发表留言
7. **AI 助手** → 聊天式交互 → 流式输出 → 多会话管理
8. **个人中心** → 预约记录 → 我的订单
9. **管理员登录** → 管理后台 → 用户管理 → 预约管理（确认/取消）
10. **注册** → 选择角色 → 正则校验演示
